Historic Nauvoo Tax Parcels Web Map

Project Description

I worked on this project under the direction of one of my professors. I helped him with this research over an eight month period. First, we had to enter all of my professor's data, using SQL, into a database that we designed. Next, we took the parcel fabric layer and digitized in the parcels that were split into new parcels at some point in time. I turned the GIS data into an interactive web map that joined together the parcels with their associated deeds and people.

Skills Used
